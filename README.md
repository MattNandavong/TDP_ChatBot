Instructions to Run FE

1. Clone/Download the repo
2. Navigate to /TDP_ChatBot/Razor_FE
3. Open the CommandLine/Terminal and run the following:
   1. npm install
   2. npm start
  
The web page should be displayed in your default web browser.
